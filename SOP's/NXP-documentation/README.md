# NXP-documentation
This repository documents my work using NXP's eIQ GenAI Flow 2.0.

The project demonstrates:

- Installation
- RAG database generation
- Running on PC
- Running on i.MX95
- Performance benchmarking
- Custom code modifications

A host PC will be required alongside the i.MX95 to generate the RAG database it will be using using existing PDF files.

A demo RAG file is provided which contains very rudimentary information. It and its input file can be found under the demo-RAG-database folder.

If you choose to use this file feel free to skip the host PC setup process, although you will still need another PC to view outputs from the board.

Please consult ```document-guide.md``` under ```original-documents``` to view the original SOPs used to set up this demo.
